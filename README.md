# CMU SEI Helm Charts

This branch is the publishing source for [https://cmu-sei.github.io/helm-charts](https://cmu-sei.github.io/helm-charts).
It is maintained automatically by [Github Actions](https://github.com/cmu-sei/helm-charts/blob/main/.github/workflows/release.yml). Do not commit to it directly.

Chart sources and documentation live on the [main branch](https://github.com/cmu-sei/helm-charts).

Add the repository:

    helm repo add sei https://cmu-sei.github.io/helm-charts
    helm repo update
